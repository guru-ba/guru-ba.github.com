Instalação

Ambiente Linux

Ubuntu (Baseado em Debian)

A instalação no Ubuntu é relativamente simples. É necessário instalar apenas um pacote de versionamento do Ruby. O mais famoso é o RVM (Ruby Version Manager). Ele será o responsável por baixar, compilar a instalar o Ruby na sua máquina.

Para instalar o RVM e o Ruby, execute os comandos abaixo:

sudo apt-get install curl
curl -L https://get.rvm.io | bash -s stable --ruby
rvm reload
rvm requirements

Esse comando irá imprimir uma lista das dependências para instalação de cada versão do Ruby (como JRuby, IronRuby ou o CRuby, que é esse que estamos instalando).

A copia dos comando da versão atual do Ruby 1.9.3 é a seguinte:

sudo apt-get install build-essential openssl libreadline6 libreadline6-dev curl git-core zlib1g zlib1g-dev libssl-dev libyaml-dev libsqlite3-dev sqlite3 libxml2-dev libxslt-dev autoconf libc6-dev ncurses-dev automake libtool bison subversion pkg-config

Ambiente Windows

Neste tópico vamos mostrar como podemos configurar o ambiente de Desenvolvimento em Ruby no Windows.

Visite a página de downloads para verificar a versão mais atual.

Para iniciarmos, vamos até a página de download do Ruby para baixar a versão mais recente, que hoje é a 1.9.3.

Durante a instalação, na tela de escolha onde serão instalados os pacotes, selecione as opções "Add Ruby executables to your PATH" e "Associate .rb and .rbw files with this Ruby installation".
